Cascading Style Sheets

CSS的英文全称为Cascading Style Sheets，通常译为层叠样式表或层级式样式表，是一种用于定义网页样式的计算机语言。细心的读者已经注意到，HTML有<style>等标签，已经可以定义网页样式了，但是HTML内置的样式繁琐且复杂，重复性较强。根据techopedia ，CSS allows developers to move styling elements to a separate HTML area for clean markup，CSS允许开发人员将样式元素移动到单独的HTML区域以进行干净标记，并且CSS structurally defines elements for headings <h1>, sub-headings <h2>, sub-sub-headings <h3>, etc，CSS从结构层面定义了标题<h1>，子标题<h2>，子标题<h3>等元素。CSS 代码一般由两部分组成：选择器(selector)与说明区（declaration block）。如图x所示，选择器用于选择适用的元素，如标题<h1>，说明区用于具体说明属性（如字体颜色、字体粗细等）及相对应的值（如黑色、加粗等）。语法结构层面，CSS说明区以花括号开始，属性与值之间为冒号，两个说明之间用分号隔开。
